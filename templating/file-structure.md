# File Structure

Expects the files in your template to look like this:

* ```about.twig``` (optional)
* ```contact.twig``` (optional)
* ```default.twig``` (required)
* ```templateName.jpg``` (required)
* ```home.twig``` (optional)
* ```metadata.json``` (required)
* ```stylesheet.less``` (required)
